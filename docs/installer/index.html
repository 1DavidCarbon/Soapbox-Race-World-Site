Installer

Soapbox Race World Community Installer

The Soapbox Installer was a community fan made windows package installer where new users would be able to install the Game Launcher with ease without having knowledge of extracting.

The creator DavidCarbon wanted more users to join in on the fun with a simple click of a button.

The development of the Game Launcher Installer was first spotted around August of 2017 during the development of Game Launcher Interface 1. The installer went up to version 1.0.137 and had stop updating the installer. A year later the installer was updated to 1.9.1.40 with no public announcement until the server error happened with this version, the installer released 1.9.1.41 (MeTonaTOR hotfix) where it was publicly announced due to MeTonaTOR was kicked out of the SBRW GitHub Repository.

The installer was down looked upon due to the issues it had brought up in early development, the Core Dev Team deemed the installer as a source of a fail point. On May 15, 2019, DavidCarbon had removed reference links to his GitHub Repository to download the installer, while new users will have to manually install the Game Launcher, the SBRW Team was more than happy to see DavidCarbon’s work slowly vanishing from the public scene.

DavidCarbon has yet to comment on the removal of the installer links.
