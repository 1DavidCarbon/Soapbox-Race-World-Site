ERROR 5

“Can’t connect to the HTTP-part of the server.”

Try to connect again. If the issue still persists, check if the server is online

ERROR 6

“Connection Lost”

Restart your game to connect to server again.

Make sure you have an internet connection to join the server.

ERROR 7

“I don’t think it works as how it should be”

Something didn’t work correctly! Try connecting again to the server to resolve issue

ERROR 10

“Invalid login-session, try again.”

Relaunch game to renew your login-session

ERROR 13

“Can’t connect to the XMPP-part of the server.”

Reconnecting to server will resolve issue

ERROR 404

“Not Found”

Reconnect to game server

EROR 500

“Server declines a data receive”

Reconnect to game server

Out of Memory

Check if your computer/laptop has enough memory to run the game.

Access violation (0xC0000005)

Caused by your computer not being able to correctly process the files & settings that it requires to run a particular program or installation.

Despite the error showing when you try and use particular pieces of software, or when you try and upgrade Windows, it has many different causes including the likes of faulty RAM problems, errors with the files of your PC and issues with the settings of your PC
