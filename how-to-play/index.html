Step 1

Just download the game launcher from our page. Yup that simple.

Step 2

Start the game launcher and Select a directory where NFS World will be installed. After this the launcher will automatically Download the game files if not Already Present.

Step 3

Create an account within the launcher (don’t worry its Free). Make sure to use a valid email and a password to remember.

*Note*
Certain servers requires a invite ticket to register. Please check the Discord server to receive one.

Step 4

Vroom Vroom. Have Fun!

Check out the Discord page to join community servers!

Need help? Join the Soapbox Race World’s Discord Server for assistance.
